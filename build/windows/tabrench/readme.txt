tabrench　(Windows 95以降)

タブレットPC向け指で行うドット絵作成ツール

・使い方
下のパレットをタップして色を選びます。
上の部分に絵を描きます。
Penメニューから、描くモードを選びます。
 Pen: 1ドットずつ描く
 Erase: 1ドットずつ消す
 Paint: 塗りつぶす.
 Spoit: 色を拾う

Paint/Spoitは一回タップすると、Penに戻ります。

画面のドット数が絵のサイズより大きい場合は、
上の矢印ボタンか、右下のプレビュー画像をドラッグして、
スクロールして編集します。

パレット右の選択中の色をタップすると、
RGBを指定して、色を選ぶことができます。

・ボタン
左から、
新規/開く/保存 Paint/Spoit スクロールボタン 編集画像切替
となっています。

・メニュー
[File]
 New: 新しい絵を始める
 New size: サイズを指定して、新しい絵を始める
 Open: 絵をファイルを開く
 Save: 絵を保存する。
　(新規の場合bmpフォルダに連番で保存されます。)
 Save as: ファイル名を指定して、絵を保存する
 Save++: ファイル名をa、b、cと名前を変えて保存。
　(アニメーションの新たなパターンを作るときに使います。)
 Quit: 終了
[Edit]
 Undo: 1つ戻す
 Shift: 絵全体を上下左右に1ドットずらす
 Mirror: 絵全体を上下、左右に反転する
 Rotate: 絵全体を90度回転する
 Option: 設定
[Pen] 描くモード
[Chrs] アニメーション用
[*] 更新あり

・設定
Edit-Optionで設定します。
[default size]
 Width: 絵の幅のドット数の初期値
 height: 絵の高さのドット数の初期値
[Screen]
Dot size: 編集画面の一つドットの大きさ
 Width: 幅の画面のドットの数
 Height: 高さの画面のドットの数
 Button: ボタンのサイズ
この設定によって編集画面のサイズが決まります。
設定は再起動後有効になります。

 256色で保存: 256色に減色して保存されます。
 PNG: bmpとともにpngファイルも生成します。

・アニメーション
dog1.bmp、dog2.bmpのように連番のファイルがあると、chrsメニューに
一覧が表示され、選ぶと編集画像を切り替えることができます。
画面右上のボタンでも切り替えることができます。
（10a.bmp、10b.bmpのような形式も可能です）

・メモ
palette.bmpで好きなパレットにできます。
