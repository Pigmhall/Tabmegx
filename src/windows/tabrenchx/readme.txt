tabrench　(Windows 95以降)

タブレットPC向け指で行うドット絵作成ツール

・設定
optionでドット絵のサイズを設定します。
widthは幅のドット数
heightは高さのドット数
dot sizeは編集画面の一つドットの大きさ
この設定によって編集画面のサイズが決まります。
設定は再起動後有効になります。

・使い方
下のパレットをタップして色を選びます。
上の部分に絵を描きます。
Penは１ドットずつ描くモード、Paintは塗りつぶしです。
絵は256色ビットマップで保存されます。

・メモ
pallete.bmpで好きなパレットにできます。
